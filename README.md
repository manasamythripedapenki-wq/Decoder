## Description

This project implements a 2:4 Decoder using Verilog HDL. The decoder converts a 2-bit binary input into four output lines. For each input combination, only one output is active. The design was tested using a testbench and verified through waveform simulation.

## Features

- Designed using Verilog HDL
- Implements 2:4 Decoder logic
- Verified using a testbench
- Waveform simulation performed
- Easy to understand and learn

## Tools Used

- Verilog HDL
- EDA Playground
- EPWave
